# scc
A Great cheat sheet and a quick reference command line tool for HTML, CSS and JS .

## Installing
- Download the project and run the install script.
`./install.sh`
## Usage
-  scc [-h]  [-html [HTML] | -css [CSS] | -js [JS] | [-rand {html,css,js}]
### Examples
 ``` scc -js array.map
```
